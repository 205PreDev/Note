# 개발 계획서 (협업 중심)

## 1단계: 프로젝트 준비 및 핵심 MVP 개발 (비회원 경험)

### **스프린트 1: 환경 설정 및 프로젝트 초기화**
- [ ] React Native 개발 환경 설정 (Node.js, Watchman, JDK, Android Studio/Xcode)
- [ ] `npx react-native init NoteApp` 명령어로 프로젝트 생성
- [ ] `GUIDELINES.md`에 따라 Git 브랜치 및 규칙 설정
- [ ] `DESIGN.md`에 따라 프로젝트 기본 폴더 구조 생성 (`src/`, `components/`, `screens/` 등)

### **스프린트 2: 노트 UI 및 기본 서식 구현**
- [ ] `react-navigation` 라이브러리 설정 및 기본 화면 내비게이션 구현
- [ ] 노트 목록 화면 UI 개발 (`FlatList` 사용)
- [ ] 노트 작성/수정 화면 UI 개발 (`TextInput` 사용)
- [ ] 기본 서식 (볼드, 이탤릭, 밑줄 등) 적용 UI 및 로직 구현

### **스프린트 3: 로컬 데이터 연동**
- [ ] 로컬 데이터베이스 라이브러리(WatermelonDB 또는 react-native-sqlite-storage) 설정
- [ ] `Note` 데이터 모델 정의 및 로컬 DB 스키마 설계
- [ ] 노트 생성 및 로컬 DB 저장 기능 구현
- [ ] 노트 목록 로컬 DB 연동 및 불러오기 기능 구현

### **스프린트 4: 핵심 기능 완성 및 비회원 경험 테스트**
- [ ] 노트 수정 기능 구현 (로컬 DB)
- [ ] 노트 삭제 기능 구현 (로컬 DB)
- [ ] 비회원 사용 시나리오 기반 통합 테스트 및 버그 수정
- [ ] 앱 아이콘 등 기본 리소스 적용

### **스프린트 5: 강화된 노트 편집 기능 (공통) 구현**
- [ ] 단락 서식 (제목, 목록, 체크리스트) UI 및 로직 구현
- [ ] 고급 텍스트 요소 (인용구, 코드 블록, 하이퍼링크) UI 및 로직 구현
- [ ] 실행 취소/다시 실행 (Undo/Redo) 기능 구현
