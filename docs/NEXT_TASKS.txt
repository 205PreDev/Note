## 모바일 필기 앱 - 다음 작업 목록

이 문서는 '모바일 필기 앱' 프로젝트의 다음 단계에서 진행할 주요 작업들을 정리한 것입니다.

*   **1. UI/UX 설계 문서 작성:**
    *   각 화면의 와이어프레임, 목업, 사용자 흐름 정의.
    *   디자인 시스템 및 가이드라인 수립.

*   **2. 데이터베이스 스키마 확정:**
    *   `TECHNICAL_DESIGN.md`의 데이터 모델을 기반으로 상세 테이블, 컬럼, 관계, 제약 조건 등 확정.

*   **3. API 명세 작성:**
    *   Flutter 앱과 Supabase/외부 AI 서비스 간의 모든 API 엔드포인트, 요청/응답 형식, 인증 방식 등 상세 정의.

*   **4. 테스트 계획서 작성:**
    *   앱의 품질 보증을 위한 테스트 전략, 유형(단위, 통합, 시스템), 테스트 케이스, 환경 등 정의.

*   **5. 개발 환경 설정 및 초기 프로젝트 구성:**
    *   Flutter SDK, IDE, Supabase 프로젝트 설정.
    *   기본적인 Flutter 프로젝트 구조 생성 및 초기화.

*   **6. (선택 사항) 추가 기능 명세:**
    *   `FUNCTIONAL_SPEC.md`에 아직 포함되지 않은 나머지 기능들에 대한 상세 명세 추가.
